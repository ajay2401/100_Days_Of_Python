# Syntax and Concepts

    1. print() -> prints the values

    2. print("""sososo""") -> Multi string literal print

    3. input() -> Takes value from the user. \n works within the input()

    4. len() -> gives you the length of the variable or string or value

# Types of error

    1. SyntaxError

    2. IndentationError
    
    3. NameError
    
    4. TypeError
    
    5. ValueError

# Notes

    1. print(x,y,z) will print values with a space
